# Technology Stack

## Core Development
- **Language**: C# 14.0+
- **Runtime**: .NET 10.0
- **Web Framework**: ASP.NET Core 10.0

## Data and Identity
- **Primary Database**: RavenDB (Document-oriented)
- **Identity Provider**: Custom RavenDB Implementation (`Aero.Identity`)
- **CMS Framework**: Piranha CMS Core

## Architecture
- **Pattern**: Decoupled, Repository-based
- **Integration**: Native RavenDB Client (Raven.Client.Documents)
